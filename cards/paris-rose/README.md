# 巴黎的绯色玫瑰

## v1.2.3

- 建筑图层放大并重新定位，更充分地填满各块地图空地。
- 牌匾缩小为纯地名标签，不再显示等级和状态；七类地块分别使用与素材下缘匹配的锚点。
- 建筑本体也可悬停和点击。悬停建筑、热区或牌匾时，整个地块升至地图最上层。
- 地图允许建筑越出边框显示，放大后的顶部素材不再被裁切。

## v1.2.2

- 领地界面从单张内嵌图改为 GitHub/jsDelivr 远程分层素材，角色卡体积降至约 288 KB。
- 农地、林地、猎场、工坊、城堡、村庄、河港各有四段外观，对应 1–2、3–4、5–7、8–10 级。
- 建筑精灵、点击热区与牌匾联动高亮。
- 支持施工、受损、繁荣和可升级四种通用覆层。
- 河港素材不带独立水面，直接贴合底图中的连续河道。

需启用 SillyTavern 酒馆助手与 MVU。领地地图需要网络访问 `cdn.jsdelivr.net`。

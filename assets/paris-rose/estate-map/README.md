# 巴黎的绯色玫瑰·领地地图分层素材

这套素材将领地界面拆成一张地形底图、七个可独立升级的建筑模块和四种通用状态覆层。前端根据模块等级选择素材，并将建筑、热区和牌匾作为同一交互组高亮。

## 等级映射

- 1–2 级：`tier-1.webp`
- 3–4 级：`tier-2.webp`
- 5–7 级：`tier-3.webp`
- 8–10 级：`tier-4.webp`

## 目录

- `map/terrain-v1.webp`：保留羊皮纸、边缘滑稽画、地形、道路、桥梁与连续河道的无建筑底图。
- `modules/<slug>/tier-*.webp`：农地、林地、猎场、工坊、城堡、村庄和河港的透明建筑图层。
- `overlays/*.webp`：施工、受损、繁荣和可升级提示。
- `manifest.json`：层级阈值、模块坐标和资源路径。

## 资源来源

所有位图均为本项目使用 OpenAI 图像生成工具定制生成，并经色键去底、分层裁切与 WebP 压缩；未引入第三方版权素材。

## 远程挂载

角色卡生成器默认从 jsDelivr 的仓库 `main` 分支读取，也可将 `ASSET_REF` 设为某个 Git 提交 SHA 以锁定资源版本。
